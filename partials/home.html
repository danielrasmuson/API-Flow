How it Works



API-Flow provides a single endpoint for all your API's. No more searching documentation, frustrating over OAuth, and updating depreciated API calls. Save time and sanity by using API-Flow.

1. Type in plain english what you want to accomplish.

Get events from meetup, eventbrite, and facebook from keyword and date range.

2. Create the flow and save the generated endpoint.

http://www.apiflow.com/user/23423423/flow/238423423

3. Call the endpoint directly from your code.

$.get("http://www.apiflow.com/user/23423423/flow/238423423", function(eventJSON) {
// your single request for eventJSOn
});
